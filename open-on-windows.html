🪟 Opening VolleyRecon on Windows

VolleyRecon is not yet distributed through the Microsoft Store. The app is delivered as a direct download from iCloud Drive. Because Windows checks that apps come from known publishers, you need to follow a few extra steps to open it for the first time.

Step 1 – Download the App

Download the Windows version of VolleyRecon from the link below:

☁️ Open iCloud Download Page
  1. Click the link above to open the iCloud Drive page in your browser. You can use any modern browser (Edge, Chrome, Firefox, etc.).
  2. Click the Download button (cloud icon ↓) next to the Windows app file.
  3. Your browser may ask whether you want to keep the file — click Keep or Keep anyway.
  4. Once the download is complete, open your Downloads folder in File Explorer.
  5. If the file is a .zip archive, right-click it and choose Extract All… to unzip it before running the installer or executable.

Step 2 – Bypass the Windows SmartScreen Warning

When you run the installer or the .exe file, Windows SmartScreen may display a blue dialog that says:

"Windows protected your PC — Microsoft Defender SmartScreen prevented an unrecognized app from starting. Running this app might put your PC at risk."

This is expected for apps not yet listed in the Microsoft Store or signed with an Extended Validation certificate. Follow the steps below to open the app anyway.

Bypassing SmartScreen

  1. Double-click the installer or the .exe file. The SmartScreen dialog will appear.
  2. Click More info (the small link in the lower part of the dialog).
  3. A Run anyway button will appear at the bottom of the dialog.
  4. Click Run anyway.
  5. If a User Account Control (UAC) prompt appears asking "Do you want to allow this app to make changes to your device?", click Yes.
  6. VolleyRecon will now install or launch normally.

Note: SmartScreen only shows this warning until enough users have run the app and Microsoft has built up a reputation for it. Once VolleyRecon is available in the Microsoft Store, this warning will no longer appear.

Step 3 – First Launch

After completing the steps above, VolleyRecon will open normally. On all subsequent launches you can simply double-click the app — no extra steps are needed.

Tip: If Windows Defender Firewall shows a network access prompt, you can safely click Allow access or Cancel — VolleyRecon does not require internet access to function. All data is processed locally on your device.

Why is this necessary?

Windows SmartScreen warns users about apps that do not yet have a code-signing certificate from a trusted Certificate Authority or have not yet built up a reputation with Microsoft. Obtaining a recognized code-signing certificate and publishing to the Microsoft Store will remove this warning in a future release.

The app itself is safe — it processes all data locally on your device and does not send any information to external servers.
