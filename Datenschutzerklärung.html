3. Datenerfassung in unserer App

Welche Daten werden erfasst?

Bei der Nutzung der Volleyball Analysis App werden folgende Daten erfasst:

  • Spielerdaten (Namen, Positionen)
  • Spielstatistiken (Punkte, Aufschläge, Annahmen, Blocks)
  • Matchdaten (Datum, Ergebnis, Gegner)
  • Technische Daten (Geräteinformationen zur Fehleranalyse)

Wofür nutzen wir Ihre Daten?

Die Daten werden ausschließlich zur Bereitstellung der App-Funktionalität verwendet:

  • Analyse und Visualisierung von Volleyballspielen
  • Statistikerstellung für Teams und Spieler
  • Verbesserung der App-Performance

4. Speicherung der Daten

Alle in der App erfassten Daten werden lokal auf Ihrem Gerät gespeichert. Eine Übertragung an externe Server erfolgt nur mit Ihrer ausdrücklichen Zustimmung.

5. Ihre Rechte

Sie haben jederzeit das Recht auf:

  • Auskunft über Ihre gespeicherten personenbezogenen Daten
  • Berichtigung unrichtiger Daten
  • Löschung Ihrer Daten
  • Einschränkung der Datenverarbeitung
  • Datenübertragbarkeit

6. Analyse-Tools und Tools von Drittanbietern

Diese App verwendet keine Analyse-Tools von Drittanbietern. Alle Analysen erfolgen lokal auf Ihrem Gerät.
